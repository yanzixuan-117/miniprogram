# 📷 图片资源说明

## 当前状态

项目中的图片引用已移除或替换为emoji图标，不会影响功能运行。

## 需要的图片资源

### 1. 默认头像（推荐）

**路径**: `miniprogram/images/default-avatar.png`

**用途**: 当用户没有头像时显示

**建议尺寸**: 200x200px

**建议样式**:
- 圆形头像
- 简单的图标或图案
- 网球拍 + 人物剪影
- 或纯色背景 + 图标

**临时方案**:
- 可以先用纯色占位
- 或使用网络图片URL

### 2. 应用Logo

**路径**: `miniprogram/images/logo.png`

**用途**: 应用图标、登录页面

**建议尺寸**: 200x200px

**建议样式**:
- 网球拍 + GO 文字
- 或网球 + 球拍图案
- 主色调：#667eea（紫色）

### 3. TabBar 图标（8个）

**路径**: `miniprogram/images/tab/`

**需要的图标**:
- `home.png` / `home-active.png` - 首页
- `video.png` / `video-active.png` - 视频
- `booking.png` / `booking-active.png` - 预约
- `profile.png` / `profile-active.png` - 我的

**规格**:
- 尺寸: 81x81px
- 格式: PNG
- 背景: 透明
- 颜色:
  - 未选中: #999999 (灰色)
  - 选中: #667eea (紫色)

## 🎨 快速解决方案

### 方案 1: 使用在线图标库

推荐网站：
- [Iconfont](https://www.iconfont.cn/) - 阿里巴巴图标库
- [Flaticon](https://www.flaticon.com/) - 扁平化图标
- [IconPark](https://iconpark.oceanengine.com/) - 字节跳动图标库

搜索关键词：
- "tennis" - 网球相关
- "user" - 用户/头像
- "calendar" - 日历/预约
- "video" - 视频
- "home" - 首页

### 方案 2: 使用 Emoji（已实现）

项目已使用 emoji 作为临时图标：
- 🏠 首页
- 🎬 视频
- 📅 预约
- 👤 我的

优点：
- 无需准备图片
- 支持所有设备
- 彩色显示

缺点：
- 样式不统一
- 不同平台显示效果不同

### 方案 3: 纯色占位

使用纯色背景 + 文字，无需图片：

```css
.avatar {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  display: flex;
  align-items: center;
  justify-content: center;
}
```
